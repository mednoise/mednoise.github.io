mednoise API Usage Guide¶

Basic Functions¶

about([header])

Provides a header and front-end interface for new users and pipeline workflows.

Manual Algorithms¶

manual_merge(filepath[, find, replace])

Combines multiple input images of the same size to yield one binary image that allows for common feature detection.

manual_find(filepath)

Offers an interface through tkinter to identify pixel coordinates and create tuple-lists that can be passed through a filter.

manual_edit(filepath, xy[, find])

Offers a manual method through which sections of input images can be silenced.

manual_primer(filepath[, find])

Creates one binary image from an imput image that allows for common feature detection.

Hotspot Algorithms¶

hotspot_complete(filepath, x, y[, find])

Processes inputted images using a hotspot algorithm, essentially acting as an intuitive paintbrush across the image.

hotspot_calculator(filepath, x, y[, find])

Calculates partition matrixes from input images, essentially divides the input image into all possiblehotspot combinations.

hotspot_analyzer([calc, filepath, x, y, find])

Analyzes partition matrixes from input images, essentially determining the clinical significance of each spot, preparing the image for isolation.

hotspot_isolator([calc, filepath, x, y, find])

Isolates and silences relevant partition matrixes from input images, essentially removing noise with unremarkable clinical significance from each image.

Branch Algorithms¶

branch_complete(filepath, x, y[, find, …])

Processes inputted images using a branching algorithm, essentially acting as an intuitive selector of a figure in the image.

branch_calculator(filepath, x, y[, find, …])

Calculates branches for inputted images using a branching algorithm, essentially acting as an intuitive selector of a figure in the image.

branch_analyzer(calc)

Analyzes branches for inputted images, essentially determining the clinical significance of each spot, preparing the image for isolation.

branch_isolator(filepath, calc[, find])

Analyzes branches for inputted images, essentially determining the clinical significance of each spot, preparing the image for isolation.
